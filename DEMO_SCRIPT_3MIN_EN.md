## 🎬 PRODUCTION NOTES

### **Technical Setup**
- **Resolution**: 1920x1080 (Full HD minimum)
- **Screen Recording Software**: 
  - Mac: ScreenFlow, QuickTime, or OBS
  - Windows: OBS, Camtasia
- **Voiceover**: 
  - Option A: Read it yourself (clear, confident delivery)
  - Option B: Use ElevenLabs AI (cloned voice, very professional)
  - Audio quality: 128 kbps MP3 minimum

### **B-roll/Stock Footage Sources**
- Google Earth: Free satellite imagery (screenshot/screen record)
- Pexels/Unsplash: Free farm/drought imagery
- Pixabay: Free agriculture videos

### **Voiceover Timing**
- Read script slowly, with natural pauses
- Aim for ~130-140 words per minute (not too fast)
- Add emphasis on key numbers: "40%", "$500M+", "95% accuracy"

### **Sound Design**
- Background: Subtle ambient tech/nature music (royalty-free)
- UI Clicks: Gentle "beep" or "whoosh" sounds (not distracting)
- Transition: Smooth crossfade between scenes (no sharp cuts)

### **Color Palette**
- Keep dashboard dark theme (already built-in)
- Overlay text: White, bold sans-serif font
- Button highlights: Bright green for "START", red for "STOP"

## ✨ KEY MESSAGING TIPS

### **DO:**
✅ Focus on OUTCOMES (40% water saved)  
✅ Show REAL DATA (weather API responses)  
✅ Demonstrate SPEED (one-click irrigation)  
✅ Tell HUMAN STORIES (farmers' lives improved)  
✅ Use CONFIDENT LANGUAGE ("Our system", "proven results")

### **DON'T:**
❌ Explain code or technical implementation  
❌ Show error messages or debugging  
❌ Use jargon without explanation  
❌ Spend time on non-critical screens  
❌ Read from a script (practice until natural)
